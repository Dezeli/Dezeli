# 🌿 Code Convention
일관성 있는 코드를 작성하자.

1. **문자열을 처리할 때는 쌍따옴표를 사용하자.**

2. **함수명, 변수명은 카멜케이스를 우선으로 작성하자.**

3. **연산자 사이에는 공백을 넣는다.**
   ```
    a+b+c+d // bad
    a + b + c + d // good
   ```

4. **콤마 다음에는 공백을 넣는다.**
   ```
    var arr = [1,2,3,4]; //bad
    var arr = [1, 2, 3, 4]; //good
   ```

5. **생성 함수의 맨 앞글자는 대문자로 한다.**

   ```
    function Person(){}
   ```
